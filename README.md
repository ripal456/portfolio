# Portfolio

A minimalist portfolio website built with HTML/CSS/JS (Tailwind + GSAP).

## Contents

- `index.html` — main page using Tailwind and GSAP animations
- `style.css` or `style-new.css` — custom styles (legacy/new)
- `script.js` — animations and interactions

## Run locally

Just open `index.html` in a browser.

## Deploy

You can host via GitHub Pages:

1. Push to `main` branch
2. In GitHub repo settings, enable Pages → Source: `main` → `/root`
